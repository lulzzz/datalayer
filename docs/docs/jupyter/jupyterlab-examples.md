[JupyterLab Examples Docs](https://jupyterlab.readthedocs.io/en/latest/developer/examples.html).

## Build

```bash
cd $DLAHOME/repos/jupyterlab
yarn run build:examples
```

## Run

cd $DLAHOME/repos/jupyterlab/examples/app
python main.py
